# one.io-examples
Examples for [one.io](https://github.com/ephoton/one.io).

## Prerequisites
- Node.js 7+
- Typescript 2+

## Usage
```bash
git clone git@github.com:ephoton/one.io-examples.git
cd one.io-examples
npm install
tsc
npm start
```
Then point your browser at http://localhost:3000
